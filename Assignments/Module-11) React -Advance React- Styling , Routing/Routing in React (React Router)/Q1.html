What is React Router? How does it handle routing in single-page applications?

React Router is a library for routing in React applications. It enables navigation within single-page applications (SPAs) by managing routes using components.

How it handles routing:

Dynamic Routing: Updates the UI based on URL changes.

Components: Utilizes BrowserRouter, Route, and Link for routing management.

State Preservation: Maintains application state during navigation without full page reloads.
